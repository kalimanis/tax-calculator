# Greek Tax Calculator

Tax calculator for Greek freelancers (Ελεύθεροι Επαγγελματίες) covering fiscal years 2025 & 2026.

## Features

- **Two tax regimes**: Μπλοκάκι (Άρθ. 12 §2στ) and Ατομική Επιχείρηση (Άρθ. 29)
- **2026 tax reform**: Updated brackets per Ν.5246/2025, child & youth reductions
- **EFKA integration**: Auto-calculated contributions by profession & category
- **Client location**: Domestic/foreign/mixed withholding per ΠΟΛ.1029/2018
- **Year & regime comparison**: Side-by-side tax obligation comparison
- **URL persistence**: Shareable calculation links via URL params
- **Dark mode** and **print-friendly** layout

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Radix UI, Recharts

## Development
